About

Silicon Valley Insults API is a public RESTful API that delivers up humorous tech-themed insults from the television show "Silicon Valley" to multiple endpoints, where users are able to request content by season, episode, and character.

  • No API key needed.
  • Some insults contain strong language.
  • Default rate limit is 100 requests per hour.
  • In-depth documentation can be found below.
  • Enjoy!

Sample Request and Response

Sample Request:

fetch("https://www.siliconvalleyinsults.com/api/insults")
.then((response) => response.json())
.then((insult) => console.log(insult));

API Endpoints

Get a random insult

Method GET
Path /api/insults

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Response format:

{
    season: ...,
    episode: ...,
    character: "...",
    insult: "..."
}

Get a random insult by character

Method GET
Path /api/insults/character/{character}
Parameters {character} is a string representing the first name of character you want an insult from

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults/character/jared")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Sample response format:

{
    season: ...,
    episode: ...,
    character: "Jared",
    insult: "..."
}

Errors and Exceptions

Insult does not exist

The insult you request may not exist. For example, you may request an insult from Jared from Season 1 Episode 3 when no such insult exists in the database. In this case, you will receive the following exception:

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults/season/1/episode/3/character/jared")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Sample response:

{
    "detail": "No insults available for season 1, episode 3, and character Jared"
}
