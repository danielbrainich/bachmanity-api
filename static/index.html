Silicon Valley Insults API

A restful API serving up the best insults from the HBO series "Silicon Valley"

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults/season/1/character/gavin")
.then((response) => response.json())
.then((insult) => console.log(insult));
keyboard_double_arrow_down

Sample response:

{
season: 1,
episode: 2,
character: "Gavin",
insult: "I hate Richard Hendricks, that little Pied Piper prick.",
}

API Endpoints

Get a random insult

Method GET
Path /api/insults

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Response format:

{
    season: ...,
    episode: ...,
    character: "...",
    insult: "..."
}

Get a random insult by character

Method GET
Path /api/insults/character/{character}
Parameters {character} is a string representing the first name of character you want an insult from

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults/character/jared")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Sample response format:

{
    season: ...,
    episode: ...,
    character: "Jared",
    insult: "..."
}
