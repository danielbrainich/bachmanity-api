Silicon Valley Insults API

A restful API serving up the best insults from the HBO series "Silicon Valley"

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults/season/1/character/gavin")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Sample response:

{
    season: 1,
    episode: 2,
    character: "Gavin",
    insult: "I hate Richard Hendricks, that little Pied Piper prick.",
}

Important info

  • There is no API key.
  • Many of these insults contain strong language.
  • If there's an insult you'd like me to add, let me know here.

Available endpoints

  • Get a random insult
  • Get a random insult by season
  • Get a random insult by character
  • Get a random insult by season and by character
  • Get a random insult by season and by episode
  • Get a random insult by season and by episode and by character

Get a random insult

Method: GET

Path: /api/insults

Parameters: n/a

Sample request:

fetch("https://www.siliconvalleyinsults.com/api/insults")
    .then((response) => response.json())
    .then((insult) => console.log(insult));

Response format:

{
    season: ...,
    episode: ...,
    character: "...",
    insult: "..."
}

Get a random insult by character

Path /api/insults/character/{character}
Parameters {character} = a string representing the first name of character you want an insult from
