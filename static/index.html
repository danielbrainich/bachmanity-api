Silicon Valley Insults API

A restful API serving up the best insults from the HBO series "Silicon Valley"

Docs

Important info

  • There is no API key.
  • Some of these insults contain strong language.
  • Rate limit is 100 requests per endpoint per hour.
  • If there's an insult you'd like me to add, let me know here.

Available endpoints

  1. Get a random insult
  2. Get a random insult by season
  3. Get a random insult by character
  4. Get a random insult by season and by character
  5. Get a random insult by season and by episode
  6. Get a random insult by season and by episode and by character

Get a random insult

GET /api/insults

Get a random insult by season

Request accepted: GET

Path: /api/insults/season/{season}

Example Request:
fetch("https://animechan.xyz/api/random/anime?title=naruto") .then((response) => response.json()) .then((quote) => console.log(quote));
Example Output:
{ anime: "Naruto", character: "...", quote: "..." }

Parameters:

  • season (int): The season for which to retrieve an insult.

/api/insults/season/{season}/episode/{episode}

Get a random insult for a specific season and episode.

GET /api/insults/season/{season}/episode/{episode}

Parameters:

  • season (int): The season for which to retrieve an insult.
  • episode (int): The episode for which to retrieve an insult.

/api/insults/character/{character}

Get a random insult for a specific character.

GET /api/insults/character/{character}

Parameters:

  • character (str): The character for which to retrieve an insult.

/api/insults/season/{season}/character/{character}

Get a random insult for a specific season and character.

GET /api/insults/season/{season}/character/{character}

Parameters:

  • season (int): The season for which to retrieve an insult.
  • character (str): The character for which to retrieve an insult.

/api/insults/season/{season}/episode/{episode}/character/{character}

Get a random insult for a specific season, episode, and character.

GET /api/insults/season/{season}/episode/{episode}/character/{character}

Parameters:

  • season (int): The season for which to retrieve an insult.
  • episode (int): The episode for which to retrieve an insult.
  • character (str): The character for which to retrieve an insult.
