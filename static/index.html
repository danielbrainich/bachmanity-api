Welcome to the Bachmanity API!

API Endpoints

Base URL: /

Get Random Insult

Endpoint: /insult
Method: GET
Description: Get a random insult from all seasons.
Response Example: 

                {
                    "insult": "s1 insult 2"
                }

Get Seasonal Insult

Endpoint: /insult/{season}
Method: GET
Description: Get a random insult from a specific season.
Parameters: season (string, e.g., "s1", "s2")
Response Example: 

                {
                    "insult": "s2 insult 3"
                }

For more details, refer to the API documentation.
